# Calculator Project Documentation

## 1. Project Overview
This project is a console-based calculator written in Python.  
It allows the user to perform four basic arithmetic operations: addition, subtraction, multiplication and division.  
The main goal of the project is to practice using Visual Studio Code, Git, and GitHub.
